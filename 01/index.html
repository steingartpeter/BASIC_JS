A szintaxis az importhoz a console-ban így működik:
Egy PROMISE-al beimportáljuk a module-t, MODUL-ként.
let PERSON = await import ('root/from/localhots/to/JSFile.js')
Ezután ezt a ha ez a module objektumot/függvényt exportál akkor egyből használhatóak azok.
Ha azonban ez egy osztályt exportál, akkor használni kell a module DEFAULT elemét, a class eléréséhez:
let p1 = new Person.default()
Ez a sor valójában a constructort hívje meg, és ezután használható lesz a p1 ben a Person osztály minden eleme.
Pl.:
p1.wrtId()
